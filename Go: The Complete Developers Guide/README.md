## Commands

```sh
% go run main.go
```

```sh
% go build main.go
% ./main
```

```sh
% go fmt
```

```sh
% go install
```

```sh
% go get
```

```sh
% go test
```

```sh
% go mod init [PROJECT_NAME]
```

## Notes

* Structs are pass-by-value, but maps are pass-by-reference
