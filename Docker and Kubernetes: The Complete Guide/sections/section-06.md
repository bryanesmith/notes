# Section 6: Creating a Production-Grade Workflow

- workflow: feature branch -> TravisCI -> main -> AWS Elastic Beanstalk 

- Install Node v8.11.3

- Generate our React app:
    ```sh
    % npx create-react-app frontend
    ```

- useful commands:
    ```sh
    % npm run start
    % npm run test
    % npm run build
    ```
